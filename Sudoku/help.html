Game rules

   Puzzle consists of a 9x9 grid containing given clues in various places. The object is to fill all empty squares so that the numbers 1 to 9 appear exactly once in each row, column and 3x3 box.


You have two options for inserting a value:
  • Use hardware keyboard to insert a value to a cell. 1-9 to fill a space, 0/delete/space to clear.
  • Touch a cell you want to insert a value, and then use application's online keyboard.



    Additional game options
  • Use New Board to start game over
  • Use Rest board to remove all inserted values from the board
  • Use Solve board to solve the current Sudolu puzzle

    If you have inserted wrong values, board wouldn't be solved, and you will get a notification about it.

  • Use Clear board to remove ALL values from the board
